Motor Speed, Measurement, and Driver Motor

➤ Block Diagram Open Loop

Pada sistem open loop berdasarkan block diagram di atas, memiliki cara kerja yaitu untuk dibandingkan antara nilai counter pada bagian A dan nilai dari goal speed/input speed yang diberikan pada komparator bagian B. Lalu dari hasil komparasi A < B,maka akan mengeluarkan out PWM dari komparator tersebut, yang dimana akan digunakan untuk mengerakkan motor driver sesuai dengan jumlah time on PWM pada periode tersebut.

➤ Coding VHDL Module Open Loop

Untuk coding VHDL module berdasarkan block diagram di atas adalah sebagai berikut :

Pada coding di atas, terdapat 4 input port dan 2 output port. Dimana 4 input tersebut adalah reset, duty_cycle_kanan, duty_cycle_kiri, dan clk. Sedangkan untuk 2 output adalah pwm_kanan dan pwm_kiri. Berikut adalah penjelasan lebih rinci mengenai masing - masing port :

● Input


- reset : untuk mereset variabel nilai counter dan variabel nilai high

- duty_cycle_kanan : sebagai nilai input speed 8-bit signed binary, yang berfungsi untuk menghasilkan PWM motor kanan yang akan
dikirim ke driver motor.

- duty_cycle_kiri : sebagai nilai input speed 8-bit signed binary, yang berfungsi untuk menghasilkan PWM motor kiri yang akan dikirim ke driver motor.

- clk : untuk mengatur jalannya sistem pada rangkaian, agar semua sistem berjalan secara bersama – sama

● Output


- pwm_kanan : sebagai nilai output yang keluar dari komparator berdasarkan perbandingan antara nilai counter dan nilai duty_cycle_kanan.

- pwm_kiri : sebagai nilai output yang keluar dari komparator berdasarkan perbandingan antara nilai counter dan nilai duty_cycle_kiri
