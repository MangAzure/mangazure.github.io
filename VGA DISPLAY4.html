Block Diagram with Ports


Block diagram explanation


This Block diagram has three main parts :


- Programmable logic device

- Video DAC ( data generator )

- VGA display


Where the programmable logic device has three modules


- VGA controller

- Frame buffer / image source

- Pixel clock enable generator






The input for this block diagram is the system clock ( input is given from the timing/ clock generator part ) which later will be converted into pixel clock and the other input is the RAM ( data input is given from camera control and pixel capture ) where the data is already located in the VGA controller.



1. Programmable Logic Device


➤ VGA Controller


The VGA controller is a programmable logic component that accomplishes the signal timing necessary to interface with the VGS monitor. The input required is the pixel clock and the image source, where areas VGA controller provide the horizontal and vertical syn signals as well as the pixel coordinated and display enable needed to produce the image at the right time.

It provides necessary timing signals to drive monitor through VGA port, VGA controllers internally generates the timing signal based on the pixel frequency ( pixel frequency could be calculated based on the refresh rate of the monitor and the number of pixel ( resolution ). VGA controller also control the flow of data through the VGA port.


VGA Connector pin and pinout



➤ Pixel Clock Enable Generator


The pixel clock was given the system clock as an input and it generates it into pixel clock where later on will be forwarded to the VGA controller and will be generated into pixel frequency.


➤ Frame buffer / Image source


This module receives the RAM from the VGA controller which pass through the EDID interface in the VGA port. Frame buffer is the memory that stores the image to be displayed ( the memory of the image here is RAM ) . The frame buffer passes this digital info into the Video DAC/ data generator based on command of the VGA controller.


2. Data Generator/ Video DAC


As the DAC is given digital RGB data from the image source/ frame buffer, DAC converts the RGB digital data and drives the VGA display with RGB analog signals at an appropriate voltage level.


3. VGA Display


Display the image resulted by the process done by the whole system.
